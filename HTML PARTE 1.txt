 AULA 009 - Links e Ancoras
 {

    tags

    a       // ancorar pode ser links externos/internos/arquivos 
    
    obs:

    (a)
    {
          ao usar a taga (a), dentro de href colocar o link externo que voce queira usar 

        abrir outra aba(<a href="https://site.com/" target="_blank"> rel="external")
        {
            recomendo que ao usar um link externo colocar junto um parametro escrito (target="_blank") para que ao clicar ele abra outra aba

            tambem pode usar o _self para continuar na mesma aba
        }

        rel=""
        {
          o rel seria para o algoritimo do google saber se o link e interno ou externo

          next indica que o link é para a próxima parte do documento atual
          prev indica que o link é para a parte anterior do documento atual
          author indica que é um link para o site do autor do artigo atual
          external indica que é um link para outro site que não faz parte do site
          nofollow indica que o site nao eh um site que voce confia
        }
    
          Outro atributo bem útil da tag de âncora é o (hreflang), que permite indicar qual é o idioma principal do site para onde o link está desviando o fluxo de navegação. 

          em links interno em href usar o atalho (ctrl + space) e selecionar a pasta, caso o arquivo esteja em outra pasta e voce queira voltar, ultilize (../(nome do arquivo))
    }
   
   para download recomendo colocar o parametro (download="(nome do arquivo)" e type="(midia type)")
   
   voce pode acessar os midia types acessando aqui: 
   https://www.iana.org/assignments/media-types/media-types.xhtml
 }

AULA 014 - TABELAS
{
      HIERARQUIA DE TABELAS (SIMPLES)
           
            TABLE = TABELA
            TR = LINHA DAS TABELAS
            TH = CABECALHO DA TABELAS
            TD = DADO DA TABELA
          

           border-collapse: collapse ou separete;  // ajunta ou separa as bordas da tabela


        ALINHAMENTO DO CONTEUDO:

             text-align: center;     // alinhamento horizontal
             vertical-align: middle; // alinhamento vertical 
        
        TABELAS GRANDES 

        TABLE
          CAPTION // legenda da tabela

          THEAD  // cabecalho
                TR, TD ,TH
          TBODY  // corpo
                TR, TD ,TH
          TFOOT  // rodape
                TR, TD ,TH
            
            escopo de titulos
            voce pode dizer para o navegador qual mensagem voce quer 
            passa com th

            basta colocar na tag de th

            scope="col" // qunado os dados do titulo estao em  baixo
            scope="row" // quando os dados do titulo estao ao lado
            scope="rowgroup" // em baixo agrupado
            scope="colgroup" // em cima  agrupado

        EFEITO ZEBRADO (cor sim cor nao em linhas)  
           basta usar um seletor personalizado chamado: 
           :nth-child(odd ou even) ou (2n ou 2n-1)

              exemplo
               tbody > tr:nth-child(odd)
                {
                    background-color: lightgray;
                }
         
         CABECALHO FIXO
           
           usar o position: relative;  // no style de tabela

           depois criar um style assim:

              thead > tr > th
                {
                    position: sticky;  // colar
                    top: -1px;
                    background-color: (mesma cor do cabecalho);
                }

          MESCLAGEM
           rowspan="" // mescla linhas
           colspan="" // mescla colunas

          AGRUPAR COLUNA
            
            para agrupar uma coluna podemos
            usar a tag:
                 
            <colgroup>
                <col class="cnome">
                <col class="csexo" span="2">
                <col class="cprof">
            </colgroup>

            colocando a tag <col> conforme o numero de colunas
            e colocando uma classe para elas 

            voce tambem pode usar a mesma config em mais de 1 coluna
            basta colocar span="numero desejado ->>"
